# 💰 Finance Dashboard

A modern and responsive **Finance Dashboard Web Application** to track income, expenses, and financial insights with a clean UI and powerful analytics.

---

## 🚀 Overview

The **Finance Dashboard** helps users manage their financial activities efficiently by providing:

* Real-time balance overview
* Income & expense tracking
* Visual analytics (charts & breakdowns)
* Smart financial insights

This project focuses on **UI/UX design + functional financial tracking system**.

---

## ✨ Features

### 📊 Dashboard Overview

* Total Balance display
* Total Income & Expenses summary
* Monthly growth indicators

### 💸 Transactions Management

* Add new transactions (Income / Expense)
* Categorize transactions (Salary, Food, Bills, etc.)
* Transaction history with filters
* Delete transactions

### 📈 Analytics & Insights

* Monthly cash flow visualization
* Expense breakdown (category-wise)
* Smart insights like:

  * Highest spending category
  * Savings rate
  * Monthly comparisons

### 🔍 Search & Filter

* Search transactions easily
* Filter by:

  * All
  * Income
  * Expense

### 🎨 UI/UX

* Modern dark theme
* Smooth gradients & glassmorphism design
* Responsive layout
